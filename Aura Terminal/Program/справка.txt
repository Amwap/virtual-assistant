В данном документе приведены самые полные сведения об обращении с терминалом
(но на самом деле я опять что - то забуду упомянуть) 

ИНТЕРФЕЙС
	В верхней части располагаются "чёлки" с обозначением открытого в 
	данный момент модуля, версии и даты сборки.
	Ниже расположено окно для сообщений ауры с очаровательным андроидом 
	из аниме "Блэйм!", временем сессии и полной датой
	Справа расположен блок для вывода вспомогательной информации
	

ВВОД ТЕКСТА
	Собирался с нуля, многое лежит под болтом, но в целом работает
	Буквы верхнего регистра отсутствуют, раскладка переключается нажатием
	на шифт, на каждой клавише по два символа для каждой раскладки 
	соответственно(подробнее в программном коде), 
	"Del" очищает поле ввода, 
	"Backspace" одиночным нажатием удаляет последнюю букву В поле ввода,
	серым текстом, выводятся подсказки и последний введёный текст.
	Зарезервированные для данного уровня команды подсвечиваются жирным.
	Стрелкамии "вправо" "влево" можно обратиться к предыдущим введёным
	командам или к тексту
	Стрелки "вверх" "вниз" перелистывают доступные команды
	"TAB" - вставляет текст из буффера обмена в поле ввода

ПРАВЫЙ БЛОК
	В шапке располагается наименование уровня и "list/all lists"
	Если список имеет нумерацию, то элемент можно выбрать по его номеру
	введя нужный номер в поле ввода(если прожать пробел, то магия
	превратит число в текст)  
	Прокрутка списка производится одиночными нажатиями на "Page Up" и
	"Page Down" на один фрейм(21 строка)

МОДУЛИ
	Аура - нейросетевой модуль(хаха, работающий на словаре и рандоме)
	 "добавь" - добавляет новый вопрос и ответы на него
	 "удали" - удаляет все ответы на предыдущий вопрос включая его
	 "скажи" - воспроизводит введёный текст голосом гуглобабы
	
	Блокнот - совершенно бесполезен, проще создать папку для текстовых документов
	 "добавь" - добавляет новый лист и даёт ему имя введёное после команды
	 "удали" - удаляет лист(и файл из архива)
	 "архив" - открывает папку с документами
	
	Проводник - открывает папки, ссылки, файлы(ярлыки)
	 "добавь" - добавляет новый путь
	 "удали" - удаляет имеющийся путь

	Селектор - модуль для классификации файлов(создавался для папки с музыкой)
	 первый уровень предлагает список добавленных дирректорий где:
	  "добавь" - добавляет новую дирректорию
	  "удали" - удаляет дирректорию
	  "открой" - открывает дирректорию
	
	 второй уровень(после выбора дирректории) разрешает добавлять теги к файлам
	 производить вывод по тегам или по части имени файла, просматривать теги для
	 данной дирректории, производить демонстрацию файла в стандартных приложениях
	  "рандом" - рандомный файл
	  "новые" - необработанные элементы
	  "теги" - демонстрация списка тегов(через этот раздел удобно выбирать теги для 
		команды "тг")
	  "повтори" - повторная демонстрация файла, если он был закрыт
	  "тг" - поиск по тегам
	  "нм" - поиск по имени

ПРОЧЕЕ
	Кастомизация фраз бота производится через файл config.txt
	Кастомизация интерфейсов через подмену картинки Interface.bmp (bmp only)
	сброс статистики производится через файл ststs()
